This example uses the Polyfill WASI implementation from https://github.com/ndesmic/wasm-cross. See also this Blogpost for more details about this implementation.

Getting stdin to work in the browser environment is non-trivial and not yet easily supported by the GitHub project. For this example I just decided to se the static stdin-input string of the Wasi-class and fill this from a call to prompt.
